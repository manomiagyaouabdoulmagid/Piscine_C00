# Piscine Codeloccol – C00

Ce dépôt contient les scripts réalisés pour l’exercice C00 : premiers pas avec Bash.

## Scripts

- `hello_world.sh` : Affiche "Hello, World!"
- `current_datetime.sh` : Affiche la date et l’heure actuelles
- `list_files.sh` : Liste les fichiers du répertoire
- `create_directory.sh` : Crée un répertoire `test_directory`
- `check_file_exist.sh` : Vérifie si un fichier existe ou non
